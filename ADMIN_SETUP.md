# Доступ к админке Django

## 1. Создание суперпользователя

Если у вас еще нет суперпользователя, создайте его командой:

```bash
python manage.py createsuperuser
```

Вам будет предложено ввести:
- Username (имя пользователя)
- Email (опционально)
- Password (пароль - будет скрыт при вводе)
- Password confirmation (подтверждение пароля)

## 2. URL админки

После создания суперпользователя, админка доступна по адресу:

```
http://your-domain.com/admin/
```

Или локально:

```
http://localhost:8000/admin/
```

## 3. Вход в админку

1. Откройте браузер
2. Перейдите по адресу `/admin/`
3. Введите username и password суперпользователя
4. Нажмите "Log in"

## 4. Управление пользователями

После входа в админку:

1. **Просмотр пользователей:**
   - Перейдите в раздел "Users" (Пользователи)
   - Вы увидите список всех пользователей с их статусами

2. **Одобрение пользователей:**
   - Выберите пользователя, которого нужно одобрить
   - В разделе "Profile" установите галочку "Admin approved"
   - Нажмите "Save"

3. **Массовое одобрение:**
   - В списке пользователей выберите нужных пользователей (чекбоксы)
   - В выпадающем меню "Action" выберите "Одобрить выбранных пользователей"
   - Нажмите "Go"

4. **Фильтрация:**
   - Используйте фильтры справа:
     - "Email verified" - фильтр по подтверждению email
     - "Admin approved" - фильтр по одобрению администратором
     - "Is staff" - фильтр по статусу персонала

## 5. Просмотр статусов

В списке пользователей отображаются колонки:
- **Email статус** - ✓ Подтвержден / ✗ Не подтвержден
- **Модерация** - ✓ Одобрен / ⏳ Ожидает

## 6. Заметки администратора

В профиле пользователя можно добавить заметки в поле "Admin notes" для внутренних пометок.

## Безопасность

- Не делитесь учетными данными суперпользователя
- Используйте сильные пароли
- Регулярно проверяйте список пользователей
- Одобряйте только проверенных пользователей
